Retrieve Invoices

Returns a list of invoice reference codes from your Tipalti account

Description

This endpoint retrieves invoice reference codes. You can optionally filter results using query parameters.

Available Query Parameters

count Number of results to return (1-100, default: 3)

Query Parameters

Number of results to return (1-100, default: 3)

Request Headers

Authorization Bearer ey****
Accept application/json
PUT https://api.tipalti.com/api/v1/invoices

Create or Update Invoice

Submit invoice data to create new or update existing invoices

Description

This endpoint accepts invoice data in JSON format. The request must include a payer name and an array of invoice objects.

Required Fields

PayerName Name of the payer organization
Invoices Array of invoice objects
RefCode Unique reference code for each invoice
InvoiceLines Array of line items with amounts
JSON Request Body

Request Headers

Authorization Bearer ey****
Content-Type application/json
POST https://api.tipalti.com/api/v1/invoices

Create Invoice

Create a new invoice in the system

Description

This endpoint creates a new invoice. The request must include payer information and invoice details. Returns the created invoice ID on success.

Required Fields

PayerName Name of the payer organization
Invoices Array with at least one invoice object
RefCode Unique reference code (alphanumeric with hyphens)
JSON Request Body

Request Headers

Authorization Bearer ey****
Content-Type application/json
DELETE https://api.tipalti.com/api/v1/invoices/:refCode

Delete Invoice

Remove an invoice from the system by its reference code

Description

This endpoint deletes an existing invoice by its reference code. This action is irreversible. Returns success confirmation or error details.

Path Parameters

refCode The unique reference code of the invoice to delete

Path Parameter

The reference code must contain only alphanumeric characters and hyphens

Request Headers

Authorization Bearer ey****
